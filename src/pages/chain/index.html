ChAInSDK
AI Pipeline Builder
Vercel AI SDK Workflow Engine

Build AI pipelines
like code, not chaos.

ChAInSDK is a lightweight orchestration layer built on top of the Vercel AI SDK. Design structured AI workflows, run multi-step prompts, and build complex reasoning pipelines with a fluent API.

Install


npm install chainsdk

Quick Example


import { google } from "@ai-sdk/google"
import ChAInSDK from "chainsdk"

const chain = new ChAInSDK({
  model: google("gemini-2.5-flash")
})

const result = await chain
  .start("blog-generator")

  .ai.text("topic", () =>
    "Generate a trending tech article topic"
  )

  .ai.text("outline", (ctx) =>
    `Create an outline for: ${ctx.topic}`
  )

  .ai.text("article", (ctx) =>
    `Write a full article from this outline: ${ctx.outline}`
  )

  .end()

console.log(result)

Core Concepts

AI Steps

Run LLM tasks directly inside your workflow using ai.text() or ai.object().

Context Engine

Every step writes to a shared context object. Future steps can reference previous outputs automatically.

Iteration

Process arrays with each() and automatically collect outputs across iterations.

Execution Hooks

Attach custom logic with execute() to integrate side-effects, logging, or tools.

Structured Object Generation


pipeline.ai.object(
  "layout",
  (ctx) => `
Generate a website layout for:
${ctx.topic}
`,
  {
    schema: LayoutSchema
  }
)

Schemas ensure AI outputs remain predictable and validated.

Looping Through Data


pipeline
  .each("layout.files")

  .ai.text("fileContent", (ctx) => `
Write the code for file:
${ctx.item.name}
`)

  .endEach()

Lifecycle Hooks


pipeline
  .onStepStart((step, ctx) => {
    console.log("Starting:", step)
  })

  .onStepEnd((step, output) => {
    console.log("Finished:", step)
  })
